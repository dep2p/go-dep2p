## 概述

DeP2P 是一个开放的开源项目，欢迎社区参与和贡献。本目录定义社区规范和贡献流程。

---

## 参与方式

| 方式 | 说明 |
|------|------|
| 报告 Bug | 在 GitHub Issues 中报告 |
| 功能建议 | 在 Discussions 中讨论 |
| 代码贡献 | 提交 Pull Request |
| 文档改进 | 修复或增强文档 |
| 测试反馈 | 测试预发布版本 |
