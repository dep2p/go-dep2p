## 标准执行

### 自动化检查

| 标准 | 工具 | 集成点 |
|------|------|--------|
| 代码格式 | gofmt | pre-commit |
| 代码规范 | golangci-lint | CI |
| 文档格式 | markdownlint | CI |

### 审查要求

| 变更类型 | 审查要求 |
|----------|----------|
| 核心代码 | 2+ 审查者 |
| API 变更 | 架构师审查 |
| 文档变更 | 1+ 审查者 |

## 与编码规范的关系

```mermaid
flowchart TB
    Standards[工程标准]
    CodingSpecs[编码规范体系]
    Code[代码]
    
    Standards -->|宏观指导| CodingSpecs
    CodingSpecs -->|具体约束| Code
```

| 层级 | 职责 |
|------|------|
| 工程标准 | 宏观原则、设计指南 |
| 编码规范 | 具体编码约束 |
