# Protocol Streams 模块

> **版本**: v1.1.0  
> **更新日期**: 2026-01-13  
> **定位**: 双向流（Protocol Layer）

---

## 模块概述

protocol_streams 提供双向流管理能力，支持流式数据传输。

| 属性 | 值 |
|------|-----|
| **架构层** | Protocol Layer |
| **代码位置** | `internal/protocol/streams/` |
| **协议前缀** | `/dep2p/app/<realmID>/streams/*` |
| **状态** | ✅ 已实现 |
| **依赖** | core_host, realm |
