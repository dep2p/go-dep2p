# DHT (Distributed Hash Table)

> **状态**: 📋 待实施
> **优先级**: P2 - 非核心功能

---

## 概述

Kademlia DHT 实现，用于去中心化的节点发现和内容路由。

---

## 实施状态

当前目录为占位目录，DHT 实现已从 v1.1.0 实施计划中移除。

**原因**:
- 依赖复杂的路由表和网络层实现
- 需要完善的 PeerRecord 签名验证机制
- 实施优先级低于 Bootstrap 和 mDNS

---

## 后续计划

DHT 功能将在后续版本中实施，当前版本可通过以下方式实现节点发现：

1. **Bootstrap 发现** - 通过引导节点发现
2. **mDNS 发现** - 局域网内自动发现
3. **手动配置** - 静态节点列表
